E2E_DVR_PlayBack_01

DVR – To verify playback feature on all recorded content.

Positive

XG1-XI3_1

XG1 and XI3 board should be Up and running in same network

 

XG1 should have one or more recordings in it.

1.TM loads DVR_agent via the test agent.

2. TM Frames the request url "http://ipaddress:8080/vldms/info/recordingurls" and makes a RPC call to the DVR_agent to get the list of recorded urls.

3. DVR_agent will send the url to XG1 and response is captured into the log file and send it to TM.

4. TM reads the log file to extract each recorded url and send it to the DVR_agent to play trhough mplayer.    

Checkpoint 1.A gstreamer symbol (+++) in e2emplayerappend.txt file.

   

Medium

DVR_agent
